# build-web-application-with-golang
Boilerplate to create a golang Web apllication using Mongodb

Requirements
============

* go v1.4 and above

Installation
============

* Clone repo into a folder
* Rename the folder to go-web-app
* `cd go-web-app`
* `glide create`
* `glide install`
* `go build`
* ./go-web-app to run the application
